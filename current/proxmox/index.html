    Tailscale DNS pve.tail404f3b.ts.net

    ネットワークブリッジ¶

    auto vmbr0                        # 外部ブリッジ（LAN）
  address 192.168.50.128/24
  gateway 192.168.50.1
  bridge-ports nic0

auto vmbr1                        # 内部ブリッジ（LXC用）
  address 10.10.0.1/24
  bridge-ports none               # 内部のみ、外部ポートなし
  post-up iptables NAT MASQUERADE # vmbr0 経由でインターネットへ
    ブリッジ サブネット 用途
    vmbr0 192.168.50.0/24 ホームルーター LAN（上位ネットワーク）
    vmbr1 10.10.0.0/24 LXCコンテナ専用内部ネットワーク（NAT）

    LXC 101: alpine-traefik¶

    リバースプロキシ + Tailscale ノード。Alpine Linux ベース。

    項目 値
    OS Alpine 3.23.3
    ディスク 2GB (rpool/data/subvol-101-disk-0)
    ディスク使用率 9% (191MB使用 / 1.8GB空き)
    CPU 1コア
    メモリ 512MB

    稼働サービス

    サービス 状態 バージョン
    traefik started (2日以上) 設定: /etc/traefik/traefik.yaml
    tailscale started (2日以上) tailscaled --socket=/run/tailscale/tailscaled.sock
    sshd started Ansible 管理用

    Traefik 設定概要

    entryPoints:
  http:   :80   → https リダイレクト
  https:  :443
  immich: :2283  # Tailscale 内部アクセス用（タイムアウト 600s）
  traefik: :8080 # ダッシュボード（insecure）

    ルーティングルール

    ファイル Host マッチ エントリポイント バックエンド
    immich-cf.yaml photos.rusticolus.dev https (:443) http://10.10.0.100:2283
    immich.yaml alpine-traefik.tail404f3b.ts.net immich (:2283) http://10.10.0.100:2283

    TLS 証明書: Tailscale 発行の alpine-traefik.tail404f3b.ts.net 証明書
