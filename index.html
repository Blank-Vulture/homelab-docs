    Homelab ドキュメント¶

    ゲーミング最適化・ゼロトラスト・将来拡張を目標としたホームラボのインフラ設計・構築ドキュメント。

    flowchart TB
    INET[インターネット] --> ONU[ONU]
    ONU --> FW[OPNsense FW]
    FW -->|SFP+ 1| CORE[CRS305 基幹SW]
    FW -->|SFP+ 2| DMZSW[CRS305 DMZ SW<br/>SwOS]
    CORE --> ACCESS[CRS328 Access SW]
    CORE --> LAB[CRS309 Lab SW]
    DMZSW --> DMZ[PVE-DMZ-1]
    ACCESS --> GAME[Gaming VLAN10]
    ACCESS --> WIFI[WiFi VLAN20/21]
    LAB --> SERVERS[Proxmox Cluster]
    LAB --> GPU[GPU PC / LLM]
